Network Topology

Network topology refers to the way a computer network is set up, essentially how the devices on the network are linked together.

There are 6 main types of network topologies:


Bus Network

A bus network refers to a network topology where all devices are required to handle each other's network packets, going to and from machines (stations) in the preferred order. While this is practical for smaller networks, when it scales up to bigger WAN networks, it quickly becomes heavily impractical and chaotic.


Ring Network

A ring network is a network topology that is shaped like a ring. It uses a single path from computer to computer, significantly bottlenecking the network speed and putting a lot of load on all computers on the network.


Star Network

A star network is a network topology where all computers connect to a single hub, aka a router. That router then delivers the packets where they need to go. While this significantly reduces the load on the network’s devices, it is still heavily impractical for big networks, and would force a heavy load on routers.


Mesh Network

A mesh network is a more efficient network topology where nodes are connected directly to each other, often routing the packets through the fastest available route. This is the topology used most by the internet. Since not all nodes have to be linked to every node, this provides a faster and more decentralized form of packet transmission.


Hybrid Network

A hybrid topology does not refer to a specific network topology, but a combination of various network topologies. For example, a star topology and a ring topology creates a star-ring topology. Most complicated networks tend to use hybrid topologies as they are the most configurable to a network’s needs.
