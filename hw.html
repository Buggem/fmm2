Networking Hardware

Networking hardware refers to any device that is capable of interfacing with a network of other devices.
These networks may contain various types of devices such as routers, switches, and servers, organised in different topologies.

The Router, the Switch, the Node, and the Server

Router

The TP-LINK Archer C1200 is a dual-band wireless router capable of up to 1.2 Gbps WiFi.

Routers store information on which devices send data packets, and to where it should send a packet next based on various routing algorithms.

Hubs/Smart Hubs

An essential component of network hardware, hubs provide a direct link into a network. Most routers have hubs attached to them, usually available through ethernet ports the back of the router.

WAPs

WAP stands for Wireless Access Point, referring to a device capable of connecting you to a computer network over radiowaves.
It is essentially an extension of a hub, allowing for yet another entrypoint to a computer network.

Switch

The D-Link DGS-105 5-Port Gigabit Desktop Switch acts as a middleman, connecting each computer to eachother.

Switches have a big advantage over hubs, a switch does mostly what a hub does, however it can identify the address a piece of information needs to go to and send it to only that device instead of routing it through all connected devices. They work by keeping a table of addresses so the problem of data collision happens much less.

Node

The LevelOne GNC-0105T connects to a computer's PCI slot and acts as a network node.

Each node refers to a device that is capable of storing, sending, and receiving information on networks, allowing a computer to communicate with other nodes.

Servers

A cluster of dedicated servers for an internet web site.

Servers are computers on networks dedicated to serving various content - such as files, HTML pages, or video game packets.
